¿Qué es el lenguaje HTML?

El lenguaje HTML no es un lenguaje de programación como muchos creíamos… es un lenguaje de Hipertexto.

El HTML, por sus siglas en inglés (HyperText Markup Language), tiene como principio esencial el uso de etiquetas (las denominadas tags) y funcionan de la siguiente manera:

<XXX> Este es el inicio de una etiqueta
</XXX> Este es el cierre de una etiqueta

¿Qué es una web?

Una Página Web, en pocas palabras, es tu pedacito de internet. Un documento elaborado en HTML, PHP o cualquier otro lenguaje, situado en un Hosting (una computadora grande sin monitor que está permanentemente conectada a la red global).

¿Cómo publicar tu web gratis?

¡Buenas noticias! Existe una forma de publicar tu web gratis: GitHub Pages. Es un servicio de GitHub que te permite subir tus páginas a la red de forma gratuita.

Pasos para publicar en GitHub Pages:

  1. 1- Crea una cuenta en GitHub (gratis).
  2. 2- Crea un repositorio llamado tusuario.github.io.
  3. 3- Sube tu archivo index.html.
  4. 4- Ve a Settings > Pages y activa la rama main.
  5. 5- ¡Listo! Tu web estará disponible en tusuario.github.io.

¿Qué es el CSS?

El CSS es la parte visual, la que engancha a los usuarios. Es el conjunto de reglas que dan color, tamaño, márgenes y diseño a tu página. ¡Es lo que hace que tu web pase de ser un documento a una experiencia única!
