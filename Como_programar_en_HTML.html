¿Qué es HTML?

HTML es el esqueleto de toda página web. Es el lenguaje que usan los navegadores para entender cómo mostrar el contenido: títulos, párrafos, imágenes, enlaces, etc.

Las siglas HTML significan HyperText Markup Language (Lenguaje de Marcado de Hipertexto).

La estructura básica de una página HTML

Toda página HTML tiene una estructura mínima que debes memorizar. Es como el "esqueleto" de tu web:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primera página</title>
</head>
<body>
  <h1>¡Hola mundo!</h1>
  <p>Esta es mi primera página web.</p>
</body>
</html>

Explicación línea por línea:

  • <!DOCTYPE html> → Le dice al navegador que es HTML5.
  • <html lang="es"> → El inicio de la página, con idioma español.
  • <head> → Aquí van los metadatos (título, descripción, etc.).
  • <title> → El título que aparece en la pestaña del navegador.
  • <body> → Todo lo que se ve en la página.
  • <h1> → El encabezado principal (solo uno por página).
  • <p> → Un párrafo de texto.

Etiquetas esenciales que debes conocer

Estas son las que más vas a usar:

  • <h1> a <h6> → Encabezados (del más importante al menos importante).
  • <p> → Párrafo de texto.
  • <a href="url"> → Enlace a otra página o sitio.
  • <img src="imagen.jpg" alt="descripción"> → Imagen.
  • <ul> y <ol> → Listas desordenadas (viñetas) y ordenadas (numeradas).
  • <div> → Contenedor para organizar el contenido.
  • <strong> → Negrita (con énfasis importante).
  • <em> → Cursiva (con énfasis).

Ejemplo práctico: Tu primera página web

Ahora vas a crear tu primera página web. Copia este código y pégalo en un archivo llamado index.html:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi primera página web</title>
</head>
<body>
  <h1>¡Bienvenido a mi página web!</h1>
  <p>Esta es mi primera página web creada con HTML.</p>
  <p>Me llamo José Manuel y estoy aprendiendo a programar.</p>
  <img src="foto.jpg" alt="Mi foto" style="width:200px;">
  <p>Sígueme en mis redes sociales:</p>
  <ul>
    <li><a href="https://www.facebook.com/profile.php?id=100075450046235" target="_blank">Facebook</a></li>
    <li><a href="https://www.instagram.com/satoo_body_666" target="_blank">Instagram</a></li>
  </ul>
</body>
</html>

¿Cómo ver tu página?

  • 1- Abre el Bloc de Notas (Windows) o TextEdit (Mac).
  • 2- Pega el código de arriba.
  • 3- Guarda el archivo con el nombre index.html.
  • 4- Haz doble clic en el archivo y se abrirá en tu navegador.
  • 5- ¡Ya tienes tu primera página web!
